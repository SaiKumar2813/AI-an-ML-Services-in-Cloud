Amazon SageMaker

Fully managed service for building, training, and deploying machine learning models. Provides a complete set of tools for data labeling, model training, and deployment.

Amazon Rekognition

Image and video analysis service for object and scene detection, facial analysis, and sentiment analysis.

Amazon Comprehend

Natural language processing (NLP) service for extracting insights and relationships from text.

Amazon Polly

Text-to-speech service that turns text into lifelike speech.

Amazon Translate

Neural machine translation service for translating text between languages.

Amazon Transcribe

Automatic speech recognition (ASR) service for converting spoken language into written text.

Amazon Forecast

Time-series forecasting service that uses machine learning.

Amazon Personalize

Service for creating personalized recommendations for content, products, and services.

Amazon Lex

Conversational interfaces for applications using voice and text.

Amazon Fraud Detector

Service that makes it easy to identify potentially fraudulent activities.

AWS DeepLens

Deep learning-enabled video camera for developers to get hands-on experience with machine learning.

AWS Deep Learning AMIs

Pre-configured Amazon Machine Images (AMIs) with deep learning frameworks like TensorFlow and Apache MXNet.

AWS DeepComposer

Service for creating music using generative AI models.

AWS DeepRacer

Reinforcement learning-powered autonomous racing car for developers to learn and experiment with RL.

AWS Gluon and TensorFlow on AWS

Libraries and tools for building and training machine learning models.

Amazon Elastic Inference

Service that allows attaching low-cost GPU-powered inference acceleration to Amazon EC2 instances.

Amazon Textract

OCR (Optical Character Recognition) service that automatically extracts text, forms, and tables from scanned documents.

AWS Panorama

Appliance and SDK for running computer vision models on existing on-premises cameras.

AWS Inferentia

High-performance machine learning inference chip designed to deliver high throughput and low latency.

Service for orchestrating multiple AWS services into serverless workflows.
